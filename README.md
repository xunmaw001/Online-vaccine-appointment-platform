# Online-vaccine-appointment-platform
这款在线疫苗预约微信小程序，为大家提供便捷的疫苗接种服务。在这里，你能查询附近接种门诊的开放时间、疫苗库存。可快速预约各类疫苗，涵盖儿童免疫规划疫苗与成人非免疫规划疫苗，如流感、HPV疫苗等。还能随时查看个人及家人的接种记录、接种计划。预约成功后，小程序会及时推送提醒消息，助你轻松安排接种，为健康“保驾护航” 。
